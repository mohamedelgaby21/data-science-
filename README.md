# Student Dropout Risk Prediction using CRISP-DM

## Project Overview
This project applies the CRISP-DM data science lifecycle to predict whether a student is likely to be at risk of dropout based on demographic information, academic background, and early Virtual Learning Environment (VLE) engagement.

## Objectives
- Identify students who are likely to be at risk of dropout
- Support early academic intervention
- Build a complete end-to-end data science pipeline
- Deploy the final model using Streamlit
